# ScreenshotBan
JSでスクリーンショットを禁止する機能を実装

# 実行方法

1. 最新の [Node.js](https://nodejs.org/en/) がインストールされていることを確かめる。
2. プロジェクトフォルダ（my-app）内で`npm start`を実行。
